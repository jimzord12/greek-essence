# Appendix B — Glossary

| Term | Definition |
|---|---|
| Custom-trip request | Structured inquiry providing enough information for Greek Essence to assess and respond to a travel-planning opportunity |
| Qualified lead | A suitable traveler with sufficient trip, party, budget, preference, timing, and contact information |
| Journey | A customizable sample route or itinerary used for inspiration |
| Experience | A travel style, theme, or desired outcome, such as honeymoon, gastronomy, or island hopping |
| Priority destination | A destination receiving a dedicated, high-quality launch page and stronger information hierarchy |
| Secondary destination | A supported destination initially grouped until dedicated content is justified |
| Quiet premium | Premium quality expressed through restraint, service, clarity, and attention rather than ostentation |
| Trust asset | Approved evidence that reduces risk, such as people, process, credentials, relationships, or testimonials |
| Primary conversion | Completion of the custom-trip request |
| Secondary conversion | Consultation or direct contact action |
| MVP | The smallest reliable product that can validate qualified demand and the human-led operating model |
| Prototype Specification | Downstream document defining UI composition, visual system, interaction, and responsive behavior |
| Technical Design | Downstream document defining implementation architecture, technologies, integrations, security, deployment, and testing |
