## 25. Four-Step Form Progress

### 25.1 Compact

Display:

- `Step X of 4`;
- current step name;
- simple progress bar;
- optional completed indicator.

Do not show four cramped labels across a narrow screen.

### 25.2 Medium/wide

Display a restrained stepper:

- numbered markers;
- step labels;
- connecting line;
- completed, current, and future states.

### 25.3 States

| State | Appearance |
|---|---|
| Future | muted marker and label |
| Current | primary marker, strong label |
| Complete | accent/primary check marker |
| Error | destructive marker plus text |
| Disabled navigation | visible but noninteractive |

Progress must use text and structure, not color alone.

### 25.4 Saved-draft state

Use a quiet information row near the form:

- small save/check icon;
- `Your progress is saved on this device for up to 72 hours`;
- `Clear saved form` as tertiary destructive action;
- saved confirmation via polite status text;
- expiry/unavailable state uses information/warning styling, not error unless data is actually lost unexpectedly.
