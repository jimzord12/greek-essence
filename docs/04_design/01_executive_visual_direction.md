
# Part I — Design Direction

## 1. Executive Visual Direction

Greek Essence should feel like an experienced, cultivated Greek host with international awareness: warm, refined, specific, calm, confident, and quietly premium.

The interface is an **editorial boutique-hospitality experience**, not a booking marketplace, SaaS dashboard, package catalogue, or luxury-fashion campaign.

### 1.1 Core direction

| Area | Decision |
|---|---|
| Overall character | Editorial boutique hospitality |
| Emotional priority | Trust first, then authenticity, calm confidence, and anticipation |
| Appearance | Light theme only |
| Color philosophy | Aegean Editorial |
| Primary canvas | Warm ivory |
| Primary brand color | Deep Aegean blue |
| Accent | Muted Aegean teal |
| Display typography | Cormorant Garamond |
| Body/UI typography | Inter |
| Density | Generous editorial |
| Corner language | Soft editorial |
| Elevation | Minimal natural shadows |
| Border language | Soft, low-contrast 1 px borders |
| Primary CTA | Solid deep-Aegean button |
| Hero treatment | Cinematic editorial |
| Header | Transparent over approved hero; solid warm ivory after scroll |
| Iconography | Lucide |
| Decorative language | Subtle Mediterranean details |
| Photography role | Primary emotional and geographic expression |
| Motion | Restrained, functional, and reduced-motion aware |

### 1.2 Brand translation

| Product principle | Visual consequence |
|---|---|
| Trust before conversion | Stable layout, strong readability, clear hierarchy, restrained effects, visible focus, no visual manipulation |
| Authenticity through specificity | Genuine place photography, accurate captions, contextual imagery, no generic “Greek” decoration |
| Quiet premium | Generous space, refined typography, controlled palette, excellent detail, little ornament |
| Human expertise is the product | Prominent people/process content, warm portrait treatment, editorial rather than corporate trust modules |
| Inspire before selling | Immersive imagery and narrative pacing before qualification-heavy surfaces |
| Progressive commitment | Calm form rhythm, clear steps, helpful text, no dense application-like UI |
| One primary action | One visually dominant deep-blue CTA treatment |
| Maintainability over novelty | Small token vocabulary, reusable visual patterns, no one-off effects |
| Honest representation | No price badges, availability chips, review stars, fake counters, or luxury claims without evidence |
