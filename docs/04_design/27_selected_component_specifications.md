## 27. Selected Component Specifications

### 27.1 C-01 Global Header

#### Anatomy

- logo;
- primary navigation;
- primary CTA;
- language switcher;
- mobile menu trigger;
- optional submenu surfaces.

#### Overlay state

- transparent;
- white logo/navigation;
- primary CTA may remain solid deep blue only if contrast and visual clarity are sufficient; otherwise use white surface with deep-blue text;
- no blur/glass;
- hero image needs safe contrast zone;
- height: approximately 80–88 px wide, 68–76 px compact.

#### Solid/scrolled state

- warm ivory background;
- slate text/icons;
- soft bottom border or `shadow-sm`, not both strongly;
- height reduces by approximately 10–15%;
- transition 160–200 ms;
- active navigation has underline, marker, or weight—not color alone.

#### Submenu

- white/ivory surface;
- 12 px radius;
- 1 px border;
- `shadow-md`;
- generous but not mega-menu density;
- clear `View all` path.

### 27.2 C-02 Footer

- use `surface-subtle` or deep-blue brand surface only if final contrast and brand review approve it;
- default recommendation: limestone/ivory surface with deep text;
- section padding: 64–96 px;
- clear group headings;
- legal and utility text uses body-sm/caption;
- no newsletter until product scope supports it;
- partner marks only when approved and attributed.

### 27.3 C-05 Hero

- display typography;
- left-aligned content preferred where imagery supports it;
- maximum text width 640 px;
- one primary CTA;
- optional secondary text link;
- no more than one eyebrow;
- controlled scrim;
- content safe area aligns to page grid;
- compact content moves toward lower third only if safe from browser UI and image crop.

### 27.4 C-11/C-13/C-14 Cards

#### Base anatomy

- media;
- optional metadata/eyebrow;
- title;
- concise summary;
- tags/route data where required;
- text link or whole-card link.

#### Base styling

- mostly borderless or 1 px border depending on context;
- card radius 16 px;
- image radius 16 px;
- content padding 20–24 px when contained;
- image-led editorial cards may use no outer box;
- hover: image scale max 1.02, border or link change, optional 1 px lift;
- focus: visible ring around interactive region.

#### Distinction

- Destination: place character and fit.
- Experience: outcome and traveler need.
- Journey: route, duration, mood, and customization.

Do not make all three visually indistinguishable.

### 27.5 C-19 Trust Credentials

- prefer narrative evidence over logo wall;
- each item includes owner/source/relationship;
- marks remain monochrome or original brand colors according to permission;
- no star ratings or numerical trust badges without approved evidence;
- use soft grouped surface and generous spacing;
- trust text must remain more prominent than decorative marks.

### 27.6 C-22 FAQ

- rows separated by 1 px border;
- minimal/no card-per-question styling;
- trigger padding 20–24 px;
- question uses title-md;
- chevron rotates subtly;
- answer uses body-md with readable measure;
- focus ring visible around trigger;
- no decorative animation.

### 27.7 C-23 CTA Block

- use deep-blue surface or warm subtle surface based on page rhythm;
- if deep-blue surface: white/ivory text, white or contrasting primary button treatment;
- if light surface: solid deep-blue primary button;
- contextual heading uses Cormorant;
- reassurance uses body-md;
- one dominant action;
- consultation remains secondary.

### 27.8 C-27 Confirmation

- narrow centered composition;
- success icon in restrained circle;
- heading in Cormorant or strong Inter depending on context;
- no full submitted payload;
- next-step list in normal text;
- optional related content below a clear divider;
- focus moves to heading.
