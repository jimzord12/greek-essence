## 14. Surface System

| Surface | Token | Use |
|---|---|---|
| Page canvas | `background` | most pages/sections |
| Editorial surface | `surface` | cards, text panels |
| Subtle grouped surface | `surface-subtle` | FAQ, trust, selected CTA/form grouping |
| Overlay surface | `popover` | menu, popover, dialog |
| Image overlay | scrim tokens | readability only |

The site should remain predominantly warm ivory. Surface changes are purposeful and should not create alternating section stripes.

Suitable secondary-surface uses:

- trust/credibility grouping;
- FAQ;
- form shells;
- final CTA blocks;
- editorial highlights;
- contact modules;
- status feedback.
